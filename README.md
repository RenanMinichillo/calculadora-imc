# Calculadora IMC

Este projeto é uma calculadora de IMC (Índice de Massa Corporal) simples.
<br>
O objetivo é calcular o IMC de uma pessoa com base no peso e altura fornecidos, oferecendo uma interpretação do resultado.

## Funcionalidades

- Entrada de peso e altura
- Cálculo automático do IMC
- Classificação do IMC com base nos resultados
- Interface simples e fácil de usar

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
